# Role Description

We are hiring a Founding Software Engineer to work on EDA tool design and the software systems around our hardware stack.

This role sits at the boundary between software and electronics. You will help build the tooling that turns hardware intent into compilable, simulatable, and manufacturable designs.

You should be comfortable working across compiler-style infrastructure, domain-specific languages, design representations, simulation pipelines, and performance-sensitive engineering problems.

# What You Will Work On

- EDA tooling for circuit and hardware design flows
- compiler and IR infrastructure for hardware-facing languages
- simulation and analysis pipelines
- numerical methods and performance-sensitive software for design validation
- developer tooling for hardware designers and internal engineering workflows

# Ideal Background

We are especially interested in candidates with experience in one or more of the following:

- EDA tools or hardware design software
- HDLs or hardware-adjacent language tooling
- compilers, parsers, IRs, or code generation systems
- numerical methods, solvers, or simulation techniques
- software for scientific computing or technical computing
- electronics, circuits, semiconductor tooling, or hardware design

# Qualifications

- Strong software engineering fundamentals and the ability to build production-quality systems
- Experience in at least one of: EDA tools, HDLs, compilers, numerical methods, or simulation systems
- Comfort working with ambiguous product and technical requirements as an early engineer
- Interest in hardware, electronics, and the design/manufacturing toolchain
- Ability to move between research-like problems and practical implementation work

# Nice To Have

- Background in electrical engineering, computer engineering, computer science, physics, or a related field
- Experience with analog or mixed-signal design flows
- Familiarity with SPICE-like simulation, optimization, or solver infrastructure
- Experience building programming languages, DSLs, or technical developer tools
